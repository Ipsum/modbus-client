Getting Started

Instructions for first time setup of your clark Sonic device using the Clark Commissioning software:

  1. Insert Jumper: A jumper is provided with the clark Sonic device in a disconnected position. Insert the jumper across the pins labeled "Commissioning" (JMP1).
  2. Power the clark Sonic device.
  3. Establish Serial Connection: If you are using a PC, wire the 3-conductor cable from the energy monitor labeled RS-485 to the model 43-119 RS-485 converter. Connect the RS-485 converter to your computer's COM port. If your computer does not have a COM port, you may have to use a RS-232 to USB bridge. This device can be purchased from Clark Solutions or a local store.
  4. Modifying clark Sonic device settings on existing RS-485 Network: It is recommended that you use your existing MODBUS master, but this Commissioning software can be used. See Networked Devices for more info.
  5. Using a MODBUS RTU master: The device may be configured and used with a MODBUS RTU master.. A table of registers and values for this purpose can be found in the clark Sonic device User Manual.
  6. Installing clark Sonic Commissioning software: The clark Sonic Commissioning software may be downloaded from the Clark Solutions website. The installation may be run via the web or saved and installed from disk on the computer to which the clark Sonic device is connected.
  7. Configuring clark Sonic device with Commissioning Software:
    • Select the COM port to which the device is connected.
    • If configuring a single device connected directly to the PC, ensure that the software toggle setting for "Jumper" is "ON"
    • If the clark Sonic device will not be used on a MODBUS RTU network, the communication settings can be left unchanged. Otherwise, set the device id, baudrate, and parity which you desire the device to have on the MODBUS RTU network.
    • On the "Unit Settings" tab, select your desired units of measure and media type.
    • Press the "Apply Changes" button on the bottom left.
    • Remove the Commissioning jumper. The clark Sonic device is now ready to connect to a MODBUS network or may be monitored via the clark Sonic Commissioning software.
