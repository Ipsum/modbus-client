Logging

This page describes how to use the clark Sonic Commissioning software in a data-logging capacity

Setup

  1. Select the "Logging" tab in the clark Sonic Commissioning software.
  2. Press the "Path" button and select the file to log to.
  3. Toggle the "Logging" button so it says "Enabled".

Use

The logging module of the clark Sonic Commissioning software saves the output from a single connected clark Sonic device each time the "Get Data" button is pressed. A timestamp is applied and the log file is formatted as a comma separated values file (CSV). This is readable by any conventional spreadsheet software such as Microsoft Excel. This behavior will continue until the clark Sonic Commissioning software is exited or logging is disabled.
